#Tic-Tac-Toe

A basic Tic-Tac-Toe game built using HTML, JavaScript, and CSS.

#How to get started

To make a move, the player will use the mouse single click to mark a space. In this version, there is no provision to undo the move. As soon as the move is done, the game proceeds to the next playerâ€™s chance. At each move, The game will show whose move (X or O) it is. When the game ends, it displays one of the following three outcomes: 1) X wins 2) O wins 3) Draw
